## Product Vision

The project is not a simple study timetable generator.

The system acts as an academic planning assistant that helps students manage changing academic goals throughout a semester.

The system supports:

* Semester planning
* Internal test preparation
* External exam preparation
* Viva preparation
* Placement preparation
* Revision planning
* Multiple concurrent goals

The system should adapt when priorities change.

---

## Core Problem

Students rarely follow a fixed study plan.

Academic life changes frequently due to:

* Internal exams
* Assignments
* Viva
* Semester exams
* Placement preparation

Most planning systems become useless when priorities change.

This system focuses on adaptive planning.

## Intelligence Rules

The system should:

* Generate study tasks
* Track completion
* Detect incomplete work
* Support goal changes
* Support multiple priorities
* Suggest plan adaptation

The system should not simply divide study hours.
